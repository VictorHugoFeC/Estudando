# Introdução à Infraestrutura de Nuvem: Descrever conceitos de nuvem
## Conceitos Básicos do Microsoft Azure
É uma plataforma de computação em nuvem. Você pode hospedar serviços Web simples para aplicativos voltados para a Internet, executar computadores totalmente virtualizados para soluções de software personalizadas ou usar serviços baseados em nuvem, como armazenamento remoto, hospedagem de banco de dados e gereciamento de conta centralizado. O Azure também oferece recursos em IA (inteligência artificial) e IoT (internet das coisas).
